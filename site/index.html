See what your codebase already has — before you build it again.

Atlas scans an entire React / Next.js / Vite / TypeScript codebase and automatically discovers, analyzes, and documents every reusable asset — components, hooks, utilities, contexts, stores, and routes. Detected by AST semantics, never by folder names.

The problem

Big frontends forget what they already built.

A hook lives in stuff.tsx next to a formatter. The same button gets rebuilt three times. New devs can't find what exists, so they reinvent it. Atlas works just as well in a tidy design system as in a messy enterprise monorepo — because it reads the AST, not your folder names.

Features

Storybook + Compodoc + Madge, auto-generated.

Everything a new developer needs to understand a codebase, produced in one scan.

🧠

Semantic AST detection

Components, hooks, utils, contexts, stores & routes classified by what the code does — resolved through the TypeScript type checker.

What it detects

Classified by AST, never by folder.

🧩 Components
PascalCase symbols that return JSX — function, arrow, forwardRef, memo & class. Props + defaultProps resolved via the type checker.
🪝 Hooks
use* symbols that don't render JSX. Params, return type, and which React / custom hooks they call.
🛠️ Utilities
Top-level functions classified as validator / formatter / helper, with purity & async inference.
🗄️ Contexts & state
createContext, Zustand create, Redux createSlice, Jotai / Recoil atom — detected by call + import origin, with state shape.
🧭 Routes
React Router, TanStack Router, and Next.js App + Pages Router — mapped to URL paths, per workspace.
How it works

A four-stage pipeline.

Load project

Resolve config, workspaces & the TypeScript program.

Extract

Run AST extractors (with cache + plugins) for every asset kind.

Analyze

Compute usage, graph, dead-code & architecture coupling.

Render

Describe, index & serve the React dashboard — or export it static.

Commands

The full toolbox.

atlas serve

Analyze and serve the interactive dashboard at a local link.

atlas export

Write a hostable static bundle (app + data.json) to deploy anywhere.

atlas analyze

Scan the project and write the JSON analysis outputs.

atlas describe

Hand assets to Claude / Codex / Cursor for richer descriptions.

atlas graph

Print the dependency graph as an ASCII tree (--json for raw).

atlas dead-code

Report unused exports, orphan files & duplicate candidates.

atlas search <query>

Fuzzy-search every discovered asset.

atlas watch

Re-analyze automatically (incrementally) on file changes.
